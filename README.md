### Tahapan Belajar

1. Pengenalan Supabase
2. Membuat Database dan Tabel Supabase
3. Konfigurasi Flutter Supabase
4. State Management BLoC
5. Menampilkan Data dari database Supabase
6. Fungsi Insert Data
7. Fungsi Centang Data
8. Fungsi Hapus Data
